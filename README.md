# :computer: Network Security  (CC4031) - 2nd Project :computer:
#### :school: Education for safety and privacy on the web
Each smartphone typically has dozens of apps, each with associated security and privacy preferences. The multiplication of associated configurations makes security/privacy unmanageable for the common user. In this work, we intend to study existing solutions for this purpose and the implementation of a mobile application in order to give security/privacy tips to the common user. Ideally, the application should check user settings and give targeted tips.
In this work, it is intended an analysis of the existing solutions for security/privacy education on the web and the development of a web application (e.g. browser extension) in order to give security/privacy tips to the common user.

#### :inbox_tray: Installation Guide :inbox_tray:
  
1.  **Open Chrome Extensions**
    > Open Google Chrome and navigate to the extensions page. You can do this by clicking the puzzle piece icon (🧩) and selecting `Manage Extensions`.

    ![Manage Extensions](https://user-images.githubusercontent.com/27723920/147799986-f8bfb234-32a7-498a-a709-a69f9e10b0cf.jpg)

2.  **Enable Developer Mode**
    > Activate "Developer mode" by clicking the toggle in the top-right corner of the extensions page.

    ![Developer Mode](https://user-images.githubusercontent.com/27723920/147800070-d4340dd1-90e5-44e1-8d12-29ef88c414df.png)

3.  **Load the Extension**
    > Click the `Load unpacked` button.

    ![Load Unpacked](https://user-images.githubusercontent.com/27723920/147800112-9d7a52fa-9486-47e8-94ce-b66613af3304.png)

4.  **Select the `src` Folder**
    > When the file dialog opens, select the `src` folder from the project's directory.

5.  **Pin the Extension**
    > The extension will appear in your toolbar. Click the puzzle piece icon (🧩) again and then click the pin icon (📌) next to the extension's name to keep it visible.

    ![Click the Icon](https://user-images.githubusercontent.com/27723920/147800340-f2116e0a-32a8-4a7f-9127-4810a734a77a.png)
    
    ![Pin the Extension](https://user-images.githubusercontent.com/20134178/147784149-02e45b35-1660-4c22-9c08-698e0d1172d8.png)

6.  **Done!**
    > The extension is now installed and ready to use.
